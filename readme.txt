1. Upload 'supreme-addons-for-beaver-builder-lite' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Beaver Builder module  Setting And Select Sabb Drag and Drop  Auto Generated Qr Code and Enter Text to Generate Qr Code.
